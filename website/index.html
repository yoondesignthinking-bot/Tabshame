Privacy

We see nothing.

  • No accounts. No emails. No server. No cloud sync.
  • Optional permissions (Recent sites, Bookmark Bar) read only local Chrome data — never us.
  • Open DevTools → Network. Watch it stay empty. That's the whole pitch.
